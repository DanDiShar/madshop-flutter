# Madshop Flutter UI

Приложение интернет-магазина на Flutter.

## Особенности
- Интерфейс интернет-магазина
- Избранные товары
- Корзина покупок
- Адаптивный дизайн

## Установка
1. Установите Flutter SDK
2. Клонируйте репозиторий
3. Запустите `flutter pub get`
4. Запустите `flutter run`
